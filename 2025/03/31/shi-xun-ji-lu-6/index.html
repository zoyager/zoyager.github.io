渗透测试实验日志 - 计算机木马制作与利用

实验环境

  • 攻击机：Kali Linux 2024.2（IP：192.168.1.100）
  • 受害机：Windows 11 22H2（IP：192.168.1.101）
  • 工具：Metasploit Framework (MSF)、Cobalt Strike（备用）
  • 网络：同一局域网（192.168.110.0/24）

实验目标

通过 Metasploit 生成远控木马并实现对 Windows 主机的远程控制。

实验步骤

1. 启动 Metasploit 控制台

2. 生成 Windows 木马

  • 参数说明
    • p: 指定 payload 类型（反向 TCP 连接）
    • LHOST: Kali 攻击机 IP
    • LPORT: 监听端口（需开放）
    • f: 输出格式（exe）
    • o: 输出文件名

3. 设置监听模块

4. 传播并执行木马

  1. 将生成的

    通过以下方式传输至受害机：

    • 共享文件夹
    • 伪装成文档 / 图片（需社会工程学）
    • 钓鱼邮件附件

  2. 在受害机上双击运行 payload.exe。

实验结果

  • 成功标志
    • msfconsole 显示 Meterpreter 会话建立
    • 可执行命令（如 shell, getsystem, upload/download）

注意事项

  1. 网络连通性：攻击机与受害机需处于同一局域网
  2. 端口开放：确保 4444 端口未被防火墙拦截
  3. 权限提升：若未获取 SYSTEM 权限，可尝试 getsystem 命令
  4. 隐蔽性：建议对木马文件进行免杀处理（如使用 veil-evasion）
